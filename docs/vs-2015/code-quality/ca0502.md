# <a name="ca0502"></a>CA0502

Desteklenmeyen dil.

Kullanıcı bir veya daha fazla sayfa veya C# veya Visual Basic dışındaki bir dilde yazılmış sınıfları içeren bir ASP.NET projesi çözümlemek çalıştığında bu hata oluşur. Bu hatayı düzeltmek için dosyayı projeden kaldırın veya Kod Analizi devre dışı bırakın.
