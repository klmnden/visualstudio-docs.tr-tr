# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28213: `_Use_decl_annotations_` ek açıklaması, değiştirilmeden önceki bildirime başvuru için kullanılmalıdır. \<neden >  
  
 Bir önceki bildirime başvuru olduğunda bu uyarı bildirilen gerekli olmadan `_Use_decl_annotations_` ek açıklama.
