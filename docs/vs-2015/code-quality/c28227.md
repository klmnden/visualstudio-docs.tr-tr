# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28227: Ek Açıklama aynı zamanda PrimOp olamaz (önceki bildirime bakın)  
  
 Bu uyarı, adlı ek açıklama sıradan bir ek açıklama olarak bildirilir ve ayrıca aynı zamanda PrimOp önceden bildirildi gösterir. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
