# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

C6995 Uyarı: XML günlük dosyası kaydedilemedi  
  
 Bu uyarı, Kod Analizi aracına Kod Analizi çıkış hata günlüğü oluşturulamıyor gösterir.  
  
 Bu hata bir disk hatasından kaynaklanıyor olabilir veya belirtilen dizine bir dosya oluşturmak için izne sahip olmayan gösterir.
