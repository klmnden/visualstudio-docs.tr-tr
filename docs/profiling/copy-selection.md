# <a name="copy-selection"></a>Seçimi kopyala
Bütün çağrı yığını raporu sekmesinden kopyalamak için tıklayın **kopyalama**. Ardından, o eylemi destekleyip herhangi bir programda çağrı yığınını yapıştırabilirsiniz.
