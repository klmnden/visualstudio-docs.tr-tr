# <a name="ca0057"></a>CA0057
Hedefleri ve kuralları geçerli kümesine ihlalleri eşlenemez.

 FxCopCmd.exe el ile bir proje dosyası ile çağrılır, bu hata meydana gelir veya bir / alınan geçerli kümesini hedefler ve kuralları için eşlenemez ihlalleri sahip XML raporu.

 Yalnızca analysis yapıldığında ortaya komut satırı aracı sürümünü kullanarak.
