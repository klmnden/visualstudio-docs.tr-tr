# <a name="c28241"></a>C28241
Uyarı C28241: Parametre üzerindeki işlev için ek açıklama tanınmıyor

 Tanınmayan ek açıklama adı kullanılır. Bu standart makroları için ek açıklamalar kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görünmesi beklenmiyor.
