# <a name="ca0051"></a>CA0051

Hiçbir kural seçilmedi.

Bu hata, yönetilen kod analizi aracı etkindir, ancak hiçbir kural seçilmedi gösterir. Bu hatayı düzeltmek için uygun kuralları'nı seçin.
