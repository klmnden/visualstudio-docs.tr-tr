# <a name="ca0503"></a>CA0503
Özelliği kullanımdan kaldırılmıştır. Önceki uygulamaların yerini alan özelliğini kullanın

 Kullanım dışı kod analizi özelliği yapı veya proje yapılandırma dosyasındaki bir değer içeriyorsa uyarı CA0503 oluşturulur. Bu uyarıyı çözmek için sürümlereni belirtilen özellik kullanın.
