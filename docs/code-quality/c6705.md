# <a name="c6705"></a>C6705
Uyarı C6705: Ek açıklama hatası < expected_number > bağımsız değişkenleri ek açıklaması için beklenen \<parametre > < actual_number > bulundu.

 Ek açıklamalarda bir hata olduğunda bu uyarı bildirilir.
