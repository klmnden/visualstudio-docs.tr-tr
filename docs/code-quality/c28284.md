# <a name="c28284"></a>C28284
C28284 Uyarı: Sembolü için koşullar işlev olmayan semboller için şu anda desteklenmiyor

 Bu uyarı bildiren bir koşullu ek açıklaması (koşul, `__drv_when`) bir işlev dışında bir şey üzerinde bulunamadı.
