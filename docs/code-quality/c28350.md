# <a name="c28350"></a>C28350
Uyarı C28350: Ek açıklama \<ek açıklama > olarak uygulanabilir olmayan bir durumu betimliyor.

 C/C++ türü burada denetlenen bir ek açıklama uygulandığında, genellikle bu uyarı oluşturulur.
