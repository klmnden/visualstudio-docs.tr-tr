# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName Metodu
Yığın çerçevesinin kullanıcı dostu adını alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pName`  
 [out] Yığın çerçevesi adı.  
  
## <a name="return-value"></a>Dönüş Değeri  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** jscript9diag.h
